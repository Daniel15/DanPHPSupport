Installation Instructions

NOTE: I haven't had time to write some proper installation instructions, so this is definitely not complete. However, it should aid you in installing DanPHPSupport. In future versions of DanPHPSupport, there will be an automatic installer, but as this is a beta version, you have to install it manually. READ THIS CAREFULLY!

1. Create the Database
You'll need to create a database for DanPHPSupport. You can do this in phpMyAdmin by using the 'Create New Database' option in it. Call the database 'support' (or something similar).

2. Import the database
The DanPHPSupport database is inside the 'install.sql' file that comes with DanPHPSupport. Use phpMyAdmin to import this SQL file (click on the database on the left side of the screen, then click on 'SQL' at the top. Choose the install.sql database using the 'Browse...' or 'Choose' button at the bottom.

3. Change the admin user settings (including password)
Unfortunately, there isn't a users control panel in DanPHPSupport (yet). So, to change your password, make sure you're still in phpMyAdmin and click on the DanPHPSupport database on the side. Click on the 'users' table and when it loads, click 'Browse' at the top (next to 'Structure'). Click on the pencil icon next to the Default Administrator. This will allow you to edit it. Change the First Name, Last Name, E-Mail and Username to whatever you want. For the password box, type in a new password and click 'MD5' in the dropdown box next to it. When you click 'Go' at the bottom, this will encrypt your password

4. Change some other settings
You also need to change a few other settings. Click on the 'settings' table and then click 'Browse' (next to Structure). Tick the 'adminEmail' and 'fromEmail' fields, and then click on the pencil icon next to 'With Selected:'. Set the following values:
- adminEmail - Your email address
- fromEmail - The email address the support notification come from (when you reply to someone's ticket). This shouldn't be a real email address (eg. make it something like support-noreply@example.com, replacing example.com with your domain)

5. Set the DanPHPSupport settings(/includes/settings.php)
You'll need to tell DanPHPSupport what your database settings are. Open the includes/settings.php file, and change the settings to suit your configuration.

6. Open the DanPHPSupport administration panel
Go to the /admin.php address, for example if you installed DanPHPSupport at http://www.example.com/support/ then you should go to http://www.example.com/support/admin.php. Add some support ticket categories and knowledgebase categories

You're done! Test out your new DanPHPSupport installation and see if it works properly.
