Installation Instructions

NOTE: If you're upgrading from DanPHPSupport 0.1 or 0.2, then see the upgrade.htm file for more information on how to upgrade

DanPHPSupport 0.3 Beta has an auto installer, so it is much easier to install! However, there are still a few things you need to do:

1. Create the Database
You'll need to create a database for DanPHPSupport. You can do this in cPanel by going to the 'MySQL Databases' option. You can also do this from within phpMyAdmin by using the 'Create New Database' option in it. Call the database 'support' (or something similar). If you created the database using cPanel, remember to create a database user and add it to the database.

2. 'CHMOD' includes/settings.php to 0777
In order to save your database settings, the DanPHPSupport installer needs to write to your settings.php file. To make the file writable, use your FTP client to 'CHMOD' it to 0777. This feature might be called any of the following: 'CHMOD', 'File Permissions', 'File Modes' or similar.

3. Run the installer!
Now, point your browser to the installer.php file, for example, if you installed DanPHPSupport at http://www.example.com/support/, then you would go to http://www.example.com/support/installer.php. Follow the instuctions within the installer.

4. Open the DanPHPSupport administration panel
Go to the /admin.php address, for example if you installed DanPHPSupport at http://www.example.com/support/ then you should go to http://www.example.com/support/admin.php. Add some support ticket categories and knowledgebase categories

You're done! Test out your new DanPHPSupport installation and see if it works properly.
