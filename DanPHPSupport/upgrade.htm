Upgrading from previous versions

To upgrade from an old DanPHPSupport version to a newer one, you need to overwrite the old DanPHPSupport files with the new ones. However, some versions may have specific upgrade instructions, such as database edits, etc. Please read below to see what you need to to do upgrade your installation to a new version.

DanPHPSupport 0.1 BETA --> 0.4 BETA

These are the instructions to upgrade DanPHPSupport 0.1 BETA to 0.4 BETA. Once you have copied the new DanPHPSupport files over your old ones, run the following queries on your database:

ALTER TABLE ticket_messages ADD FULLTEXT (message);
INSERT INTO settings (field, value) VALUES ('timeZone', '10');
CREATE TABLE `ticket_canned` (
    `ID` TINYINT UNSIGNED NOT NULL AUTO_INCREMENT ,
    `name` VARCHAR( 80 ) NOT NULL ,
    `text` TEXT NOT NULL ,
    PRIMARY KEY ( `ID` ) 
);

You may do this in phpMyAdmin by choosing the database (on the right-hand side), and then clicking on 'SQL' at the top of the page.

DanPHPSupport 0.3 BETA --> 0.4 BETA

This is really easy: Just copy all the new files to your installation directory, overwriting the old files. That's all!
